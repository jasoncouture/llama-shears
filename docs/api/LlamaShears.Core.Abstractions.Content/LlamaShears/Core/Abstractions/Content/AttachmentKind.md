# LlamaShears.Core.Abstractions.Content.AttachmentKind

Assembly: `LlamaShears.Core.Abstractions.Content`

Modality of a non-text payload attached to a turn. Today the
framework recognizes only [AttachmentKind](AttachmentKind.md).`Image`; additional kinds
will be added as providers gain support.
