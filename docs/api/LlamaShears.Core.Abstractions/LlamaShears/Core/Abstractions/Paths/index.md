# LlamaShears.Core.Abstractions.Paths

## Types

- [IShearsPaths](IShearsPaths.md) — Resolves on-disk paths for the well-known categories of host state ([PathKind](PathKind.md)). Implementations decide where each root lives and whether to create directories on demand.
- [PathKind](PathKind.md) — Well-known categories of host state whose on-disk root is resolved by [IShearsPaths](IShearsPaths.md). Implementations decide where each root lives and whether to create directories on demand.
